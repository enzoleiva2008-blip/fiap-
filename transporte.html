Integração técnica

Como a solução se integra ao sistema de transporte

Dois modelos de integração para diferentes contextos e operadoras.

Modelo Voucher Digital

O modelo mais simples: o usuário gera um QR Code no app SoulUp que é lido pelas catracas equipadas ou validado pelos agentes de transporte.

📱

Geração no app

Usuário seleciona pontos, escolhe o modal e gera o QR Code com validade de 24h.

🔍

Validação na catraca

Leitor da catraca comunica com a API SoulUp para validar e consumir o voucher.

✅

Confirmação instantânea

O sistema registra a viagem, debita os pontos e atualiza o histórico em tempo real.

✅ Vantagens deste modelo

  • Implementação mais rápida
  • Não requer alteração nos sistemas legados
  • Funciona offline (QR salvo em cache)
  • Rastreabilidade completa da viagem
  • Compatível com qualquer operadora

📊 Anti-fraude: Token único por CPF, expiração automática em 24h e limite de 3 tentativas de uso por QR Code.

Integração via API

Para operadoras com API disponível (SPTrans, CPTM), a integração é direta: os créditos são adicionados ao cartão de transporte existente do usuário.

🔗

Autenticação OAuth 2.0

Usuário autoriza a SoulUp a acessar seu cartão de transporte cadastrado na operadora.

💳

Recarga direta no cartão

A API da SoulUp debita os pontos e credita viagens diretamente no cartão (ex: Bilhete Único).

📊

Relatório consolidado

Histórico unificado de pontos usados e viagens realizadas, com dados ambientais calculados.

🔧 Dados técnicos

Protocolo REST / HTTPS
Autenticação OAuth 2.0 + JWT
Formato JSON
Latência < 500ms
Disponibilidade 99,9% SLA
Segurança

Sistema anti-fraude

Cada etapa do processo tem camadas de segurança para garantir a integridade das transações.

🔐

Autenticação biométrica

Face ID ou impressão digital obrigatória para gerar qualquer voucher de transporte.

🎫

Token único

QR Code gerado com hash único, vinculado ao CPF e ao dispositivo. Uso único e irrevogável.

⏰

Expiração automática

Vouchers expiram em 24 horas. Tentativas de reuso são bloqueadas e registradas.

🤖

Detecção por IA

Modelo de machine learning monitora padrões anômalos de uso e bloqueia contas suspeitas.
